# 如何遊玩 (How to Play)

開啟專案後，你可以直接選擇開始遊戲，預設將會是 3*3 的拼圖，
或是你可以在第一個頁面，輸入n (n * n) 的矩陣，
並在下一個頁面中，選擇一個你喜歡的位置，當作空格，
就能開始遊戲。

開始遊戲後，請耐心等待一段時間，
演算法的運算結束後；就能開始遊戲。

除了手動點選空格周圍的方塊交換位置外，
你也可以使用A* 算出來的 "演算法下一步"；
或是我們自己寫的評估AI，預測出來的下一步最佳解，
當然你也可以讓她一次解完，連動都不用動。

遊玩過程中，可以儲存紀錄，
再下次打開遊戲時載入他們。

最後，你可以在一開始的畫面，選擇你喜歡的圖片，
作為拼圖的底圖，或是就用數字方塊開始解謎。
